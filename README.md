# Do-not-let-ball-down

Hey! Welcome to my physics drop puzzle game. This game was a great way to practice handling coordinate vectors and custom geometry collisions in a 2D environment. It is just a fun little project, but I think it works pretty well!

## Controls

* **Draw Lines:** Left Click and Drag the mouse
* **Exit Game:** Escape Key

## Why I built it

I actually built this because I saw my younger sister playing a game just like it on her phone. I told her, "I can make you a game exactly like that!"—so I sat down, coded it up, and here it is. It turned out to be a great way to push my logic skills, especially when figuring out how to prevent fast-moving objects from breaking through custom drawn walls. 

## What was used to build it

I used Python's pygame library to manage the window rendering and game loop, along with standard math modules to handle vector projections and circle-to-line segment physics.
